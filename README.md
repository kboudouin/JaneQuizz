# JaneQuizz

node -v : v18.15.0
To start to project run these following commands:

In front-quizz :
npm install
npm run dev

In back-quizz:
uvicorn main:app --reload

---- DOCKER ----
In JaneQuizz directory run :
docker-compose up --build

FRONT URL : http://localhost:3000
BACK URL : http://0.0.0.0:8000
